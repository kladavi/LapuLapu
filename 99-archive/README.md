# 99-archive

Historical or retired material. Preserved for traceability; **not** part of
active operations.

## Rules

- Move, don't delete. Files retired from `02-work/` or `03-reporting/` land
  here with their original relative path when practical.
- Archived files are **read-only in intent**: do not edit content; add a
  short header note if context is needed.
- The focus generator and reporting scripts **should not** scan this folder.
- If something needs to come back to life, copy it forward into the active
  tree; leave the archived copy in place.
