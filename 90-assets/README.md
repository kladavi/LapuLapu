# 90-assets

Non-text supporting material: images, screenshots, PDFs, diagrams, exported
decks, attachments referenced from markdown elsewhere in the repo.

## Conventions

- Reference assets from markdown with **relative paths** (e.g.
  `../90-assets/foo.png`) so links survive moves.
- Prefer descriptive filenames: `2025-11-14-matryoshka-arch.png`, not
  `image1.png`.
- Keep binaries reasonable in size; large PDFs belong in an external store,
  linked from here.
- Do **not** put source-of-truth content here — this folder is for support
  material only.
